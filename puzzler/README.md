Puzzle-R é um jogo de quebra-cabeça desenvolvido em Python 3.8 utilizando as bibliotecas Tkinter e Pygame.
Há possibilidade de importar suas próprias imagens e o jogo possui 3 níveis de dificuldade,sendo eles 
fácil, médio e difícil.
Para bater um record, deve-se completar o quebra-cabeça com menos movimentos possíveis.
OBS: Também foi feito o uso das bibliotecas Pillow e Numpy para edição de imagem e utilização
de matrizes.
